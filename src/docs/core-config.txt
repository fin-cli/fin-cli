## OPTIONS

* `--dbname`=<dbname>:

	Set the database name.

* `--dbuser`=<dbuser>:

	Set the database user.

* `--dbpass`=<dbpass>:

	Set the database user password.

* `--dbhost`=<dbhost>:

	Set the database host. Default: 'localhost'

* `--dbprefix`=<dbprefix>:

	Set the database table prefix. Default: 'wp_'
